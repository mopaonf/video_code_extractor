# Video Code Extractor App

## How to Run the App

1. **Navigate to the project directory**:
   ```
   cd d:\notes\Path_to_C\Django\code_extractor\video_code_extractor_app
   ```

2. **Install dependencies** (if not already installed):
   ```
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```
   python main_app.py
   ```

## How to Test the App

1. **Run the application** and process a video file to extract code snippets.
2. **Verify functionality**:
   - Check if the extracted snippets are displayed in the app.
   - Use the filter and export features to ensure they work as expected.

## Notes
- Make sure Python is installed on your system.
- If you encounter issues, check the console for error messages and resolve them accordingly.
